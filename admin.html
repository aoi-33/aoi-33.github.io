Fine-grained Personal Access Token を使い、data/entries.json をブラウザから直接コミットします。 トークンは このブラウザ（localStorage）にのみ 保存され、GitHub API 以外には送信されません。

—

トークンの作り方
  1. Fine-grained PAT 発行ページを開く
  2. Repository access → Only select repositories でこのリポジトリだけを選択
  3. Permissions → Repository permissions → Contents: Read and write
  4. 発行されたトークンを上の欄に貼り付け

※ 権限をこのリポジトリの Contents だけに絞れば、万一漏れても被害はこのリポジトリのファイル編集に限定されます。
